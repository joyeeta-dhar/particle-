# 3D Particle Morpher with AI Hand Tracking

An immersive, interactive 3D particle system built with **Three.js** and **MediaPipe Hands**. This application uses your webcam to track your hand movements and gestures in real-time, allowing you to morph a cloud of 4,000 particles into various geometric and organic shapes.

![Particle Morpher](https://raw.githubusercontent.com/joyeeta-dhar/particle-/main/index.html) *(Note: Replace with actual screenshot later if needed)*

## ✨ Features

- **🦾 Real-time Hand Tracking**: Powered by MediaPipe's robust hand landmarks model.
- **🌀 Dynamic Morphing**: Smooth transitions between complex 3D shapes.
- **✨ 4,000 Particles**: High-performance rendering using Three.js BufferGeometry.
- **🎨 Interactive Color & Rotation**: Hand position controls the particle hue and global rotation.
- **🔭 Pinch to Zoom**: Control the scale of the particle system with a pinch gesture.

## 🖐 Controls & Gestures

Hold up different numbers of fingers to switch between these shapes:

| Gesture | Shape |
| :--- | :--- |
| 🖐 **1 Finger** | ❤️ **Heart** |
| 🖐 **2 Fingers** | 🪐 **Saturn** (Sphere + Ring) |
| 🖐 **3 Fingers** | 🌸 **Flower** (Rose Curve) |
| 🖐 **4 Fingers** | 🧬 **DNA Helix** |
| 🖐 **5 Fingers** | 🌕 **Sphere** |

- **👌 Pinch (Thumb + Index)**: Expand or contract the particle system.
- **↔ Move Hand**: Rotate the system and transition colors.

## 🚀 Tech Stack

- **Three.js**: 3D graphics engine.
- **MediaPipe Hands**: ML-based hand tracking.
- **Vanilla JavaScript**: Logic and interaction.
- **CSS3**: Premium dark-mode UI.

## 🛠 Setup & Usage

1. Clone the repository or download the `index.html` file.
2. Open `index.html` in any modern web browser (Chrome, Edge, or Brave recommended).
3. **Allow camera access** when prompted.
4. Position your hand in front of the camera and start morphing!
